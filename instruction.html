Instruction for Printing Certificates


1. Make sure you are in the right certificate page. Enter Date and Name


2. Add names or remove the names


3. Once you check all the information is correct, Click the "Submit" button.


4. Review the certificates dispaly below.


5. If you decide not to print them, Click the "Reset" button.


6. If all the information is correct, Click "Print" button. It will take you to a new page.


7. Right click the page, select "Print" to print out the Certificate.

* When the printing preview window pops out, make sure you select "Landscape" for the layout and adjust the margins if it's needed.
